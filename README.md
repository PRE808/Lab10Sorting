# Generic Sorting Algorithms in Java

This project demonstrates two generic sorting algorithms implemented in Java.

## 🔹 Features

* Insertion Sort using Comparable
* Bubble Sort using Comparator
* Works with multiple data types (Integer, String)

## 🧠 Concepts Used

* Java Generics
* Comparable Interface
* Comparator Interface
* Sorting Algorithms

## ▶️ Example Output

Insertion Sort:
1 2 3 5 9

Bubble Sort:
Alice Bob John

## 🚀 How to Run

1. Open the project in NetBeans
2. Run SortingLab.java

## 📚 Purpose

Created as part of a lab assignment to practice generic methods and sorting algorithms.
